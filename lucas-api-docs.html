LUCAS API

The LUCAS API provides programmatic access to physical asset verification. Request inspections, retrieve cryptographically signed proofs, and integrate real-world condition data into your smart contracts.

Base URL

https://api.lucas.app/v1

Verified Data

GPS-stamped, cryptographically signed

On-chain Ready

Anchor to any EVM chain

Quick Start 
curl https://api.lucas.app/v1/proofs \
  -H "Authorization: Bearer lcs_live_xxx"import Lucas from '@lucas/sdk';
const lucas = new Lucas('lcs_live_xxx');
const proofs = await lucas.proofs.list();import lucas
client = lucas.Client('lcs_live_xxx')
proofs = client.proofs.list()

Authentication

Include your API key in the Authorization header.

API Key Format

lcs_live_xxxxxxxxxxxxxxxx

Protect your API keys

Never expose keys in client-side code.

Authenticated Request 
curl https://api.lucas.app/v1/proofs/PROOF-2024-0847 \
  -H "Authorization: Bearer lcs_live_xxxxxxxxxxxx"

Proofs

A Proof is a cryptographically signed record of a physical asset's condition at a specific point in time.

GET /proofs List all proofs
GET /proofs/:id Retrieve a proof
POST /proofs/request Request new proof

The Proof Object

proof_idstringUnique identifier
asset_idstringRelated asset
condition.scoreintegerScore 0-100
issuesarrayDetected issues
signature.hashstringSHA-256 hash

Webhooks

Receive real-time notifications when events occur.

Event Types

proof.completed New proof ready
proof.anchored Written to chain
issue.detected New issue found
